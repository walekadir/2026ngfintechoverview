DECODE: Nigeria Fintech Landscape 2026

An interactive data visualization report tracking the evolution, players, and consolidation of the Nigerian Fintech ecosystem as of April 2026.

 🚀 Overview
This project is a high-fidelity, single-page intelligence report designed to simplify the complex Nigerian fintech market. It tracks over 430+ companies, 4 unicorns, and the massive shift from simple payment processing to full-scale financial infrastructure.

Features
- Company Radar: Filterable directory of top-tier players (Unicorns, Neobanks, Infrastructure).
- Interactive Timeline:A decade of disruption from 2015 to 2026.
- Market Analytics: Visualized data on valuations, transaction volumes, and annual funding.
- M&A Tracker: Mapping the "Great Consolidation" wave in African tech.
- Intelligence Signals: Deep-dive insights into Stablecoins, SME layers, and the "Infrastructure War."

Built With
- HTML5 (Semantic structure)
- CSS3 (Custom animations, grain filters, and responsive grid)
- Vanilla JavaScript (Intersection Observers for scroll reveals and dynamic data rendering)
- No Dependencies: Zero external libraries for maximum performance.

Usage
Simply clone the repository and open `ng-fintech.html` in any modern web browser.
